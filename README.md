# Yandex ML Training 2023 | Решения практических задач

В данном репозитории собраны мои решения задач в рамках программы **"Тренировки по ML" от Яндекса**. Курс был посвящен глубокому изучению алгоритмов машинного обучения, их реализации "под капотом" и применению современных библиотек для решения индустриальных задач.

---

## 📌 О проекте
Программа обучения охватывала полный цикл разработки ML-решений: от анализа данных и генерации признаков до построения сложных ансамблей и работы с глубоким обучением.

## 🛠 Реализованные темы и задачи

### 1. Линейные модели и Регуляризация
* Реализация линейной и логистической регрессии.
* Работа с переобучением: L1 (Lasso) и L2 (Ridge) регуляризация.
* Оптимизация методом стохастического градиентного спуска (SGD).

### 2. Решающие деревья и Ансамбли
* Построение решающих деревьев, критерии информативности (Gini, Entropy).
* Ансамблирование: **Random Forest** и **Bagging**.
* Градиентный бустинг: практическое применение **CatBoost**, **XGBoost** и **LightGBM**.

### 3. Метрики и Валидация
* Выбор и интерпретация метрик: ROC-AUC, PR-кривая, F1-score, Logloss.
* Настройка схем валидации (K-Fold, Stratified Split).

### 4. Обработка данных (Feature Engineering)
* Обработка категориальных признаков (One-Hot, Target Encoding).
* Работа с выбросами, пропусками и нормализация данных.
* Генерация новых признаков на основе бизнес-логики.

### 5. Введение в Deep Learning
* Основы полносвязных нейронных сетей на **PyTorch**.
* Функции активации, обратное распространение ошибки (Backprop) и оптимизаторы.
